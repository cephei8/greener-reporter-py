# Greener Reporter - Python

[![PyPI version](https://img.shields.io/pypi/v/greener-reporter.svg)](https://pypi.org/project/greener-reporter/)

Python library for implementing reporters for [Greener](https://github.com/cephei8/greener).<br>

Used by pytest plugin for Greener: [pytest-greener](https://github.com/cephei8/pytest-greener).
